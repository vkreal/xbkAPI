xbkToolBar

The xbkToolBar class provides a horizontal or vertical tool bar system.


Since: 1.0
Version: 1.0
Extends: xbkLayer

Constructor

xbkToolBar(String inst, xbkLayer parent)

    The name or id to be use for this xbkToolBar.
  • xbkLayer parent
    The parent of the xbkToolBar object if it has one.
