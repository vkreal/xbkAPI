xbkTreeNode

The xbkTreeNode data structure holds information about an item of a xbkTree.


Since: 1.0
Version: 1.2
Extends: xbkObject

Constructor

xbkTreeNode(Boolean bFolder, Reference fHandler, xbkTree xbkTreeRef)
  • Boolean bFolder
    If this xbkTreeNode is a folder or file.
  • Reference fHandler
    A reference to a handler function for this xbkTreeNode.
  • xbkTree xbkTreeRef
    A reference to the xbkTree that created this xbkTreeNode.
    •

    Methods

    hidden(Boolean bHide)
    Hide a folder or file, default to false.
  • Boolean bHide

    • Returns: Nothing
    addChildNode(xbkTreeNode xbkNewNode)
    Add a xbkTreeNode.
  • xbkTreeNode xbkNewNode
    A xbkTreeNode to add.

    • Returns: Nothing
    removeChildNode(String id)
    Add a xbkTreeNode.
  • String id
    Id of a xbkTreeNode to remove.

    • Returns: Nothing
