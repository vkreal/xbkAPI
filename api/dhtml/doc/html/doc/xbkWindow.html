xbkWindow

The xbkWindow Class creates a window like look and feel tool for any xbkAPI web applications.


Since: 1.0
Version: 1.1
Extends: xbkLayer

Constructor

xbkWindow(String inst, xbkLayer parent)
  • String inst
    The name or id to be use for this xbkWindow.
  • xbkLayer parent
    The windows parent, if any.
