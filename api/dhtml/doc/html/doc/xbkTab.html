xbkTab

The xbkTab class provides Tab Seletion options for xbkAPI web applications.


Since: 1.0
Version: 1.0
Extends: xbkLayer

Constructor

xbkTab(String inst, xbkLayer parent)
  • String inst
    The name or id to be use for this xbkTab.
  • xbkLayer parent
    The parent of the xbkTab object if it has one.
